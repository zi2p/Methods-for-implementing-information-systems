Одна из дисциплин ИСБ (ИТМО).

## Лабораторная работа 1:

### Первая часть задач: 

1.	Провести сравнительный анализ стоимости, длительности внедрения и др. параметров 6 основных ERP. 
2.	Привести обоснование выбора лучшей и худшей ERP по критериям.
3.	Подвести итоги (выводы) по анализу.

### Вторая часть задач: 

1.	Сформировать ТОП-3 из систем для выбранного объекта и обосновать выбор 
2.	Описать структуру предприятия, для которого выбирают ERP систему.
